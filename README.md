## 📖 About

This repository is the central version hub used by **SimpleDevelopments** resources.

Each supported script has its own version file containing information such as the latest available version, download or release location, and changelog history. Resources can check these files when they start and notify server owners when a newer version is available.

Keeping this information in one repository makes it easier to manage version checking across multiple SimpleDevelopments projects without hardcoding update information into every individual resource.

This repository is mainly used internally by SimpleDevelopments and is not intended to be installed as a standalone resource.

---

## 📂 What Is Stored Here

Most files in this repository belong to a specific SimpleDevelopments resource and contain:

- The latest available version
- The download or release URL
- Current and previous changelog information

When one of our resources is updated, its matching version file is updated here so existing installations can detect the new release.
